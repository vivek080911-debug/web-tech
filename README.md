# 🚀 Web-Tech Project

This project contains a **Frontend** and **Backend** application.

---

## 📂 Project Structure

project/
 ├── frontend/
 ├── backend/
 └── README.md

---

## 🖥️ Frontend Setup

### 📁 Go to frontend folder
cd frontend

### 📦 Install dependencies
npm install

### ▶️ Run frontend
npm start

### 🌐 Runs on
http://localhost:3000

---

## ⚙️ Backend Setup

### 📁 Go to backend folder
cd backend

### 📦 Install dependencies
npm install

### ▶️ Run backend
npm start

### OR (for development)
npm run dev

### 🌐 Runs on
http://localhost:5000

---

## 🔑 Environment Variables

Create a `.env` file inside the **backend** folder:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

---

## 🛠️ Requirements

- Node.js (v14 or higher)
- npm or yarn
- MongoDB

---

## ⚠️ Common Issues

### Port already in use
Change the port in `.env`

### Node modules error
rm -rf node_modules
npm install

---

## ▶️ How to Run Full Project

1. Start backend:
cd backend
npm start

2. Start frontend:
cd frontend
npm start

---

## ✨ Notes

- Run backend first, then frontend
- Ensure MongoDB is running
- Check API URLs in frontend code
